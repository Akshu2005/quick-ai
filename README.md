# ⚡ QuickAI – AI Powered SaaS Platform

QuickAI is a modern AI-powered SaaS web application that provides intelligent AI tools for content generation, conversations, and productivity enhancement using advanced AI APIs.


🔗 **GitHub Repository:**
https://github.com/Akshu2005/quick-ai

---

# 🚀 Features

* 🤖 AI Chat Assistant
* ✍️ AI Content Generation
* 🧠 Smart AI Responses
* 🔒 Authentication System
* 💻 Modern Responsive UI
* ⚡ Fast Vite + React Frontend
* 🌐 REST API Integration
* ☁️ Deployment with Vercel
* 📱 Mobile Responsive Design

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash id="z4x8tm"
git clone https://github.com/Akshu2005/quick-ai.git
```

---

## 2️⃣ Move into Project

```bash id="s7v9qp"
cd quick-ai
```

---

# ▶️ Frontend Setup

```bash id="w2m5ka"
cd client
npm install
npm run dev
```

Frontend runs on:

```bash id="k8r1nv"
http://localhost:5173
```

---

# ▶️ Backend Setup

Open another terminal:

```bash id="f6u3pl"
cd server
npm install
npm start
```

Backend runs on:

```bash id="d9q7rt"
http://localhost:3000
```

---

# 🔑 Environment Variables

Create `.env` files in both frontend and backend.

Example:

```env id="m1x6we"
MONGODB_URI=your_mongodb_url
GEMINI_API_KEY=your_api_key
PORT=3000
```

---

# 🌐 Deployment

## Frontend Deployment

Deploy `client` folder on Vercel.

Build Settings:

* Framework: Vite
* Root Directory: `client`
* Build Command: `npm run build`
* Output Directory: `dist`

## Backend Deployment

Deploy backend separately on:

* Render
* Railway
