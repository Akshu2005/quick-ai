# 🚀 Quick AI – Full Stack AI SaaS Platform

Quick AI is a full-stack AI-powered SaaS application that enables users to generate content, create blog titles, generate images, remove backgrounds, and more using advanced AI tools.

LIVE LINK : https://quick-ai-ochre-theta.vercel.app/
## ✨ Features

* 🔐 **User Authentication**

  * Secure login/signup (Clerk/Auth integration)

* 📝 **AI Content Generation**

  * Write articles using AI
  * Generate blog titles instantly

* 🖼️ **Image Tools**

  * AI Image Generation
  * Background Removal
  * Object Removal

* 📊 **Dashboard**

  * Track total creations
  * View recent activities

* 💳 **Subscription System**

  * Free & premium plans
  * Usage tracking

---

## 🛠️ Tech Stack

### Frontend:

* React.js (Vite)
* Tailwind CSS

### Backend:

* Node.js
* Express.js

### Database:

* PostgreSQL (Neon DB)

### Other Tools:

* Clerk (Authentication)
* OpenAI API (AI features)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/Akshu2005/quick-ai.git
cd quick-ai
```

---

### 2️⃣ Setup Frontend

```
cd client
npm install
npm run dev
```

---

### 3️⃣ Setup Backend

```
cd server
npm install
npm start
```

---

### 4️⃣ Environment Variables

Create a `.env` file inside `server/`:

```
DATABASE_URL=your_neon_postgresql_url
OPENAI_API_KEY=your_openai_key
CLERK_SECRET_KEY=your_clerk_key
```
